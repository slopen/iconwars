# iconwars

* react
* react-router
* reactstrap
* express


### install

```
$ git clone git@github.com:slopen/iconwars.git
$ cd iconwars
$ yarn
```

### production

```
$ yarn run production
```

* open `http://localhost:9020`

### development

```
$ yarn run dev
```

* open `http://localhost:9021`
